Ajuda

Para fechar o programa definitivamente, clique no X na barra superior da janela, ou clique com o botão direito no ícone do system tray e clique em "Fechar definitivamente". Além disso, clicar com o botão direito do mouse no botão flutuante da área de trabalho fechará o programa definitivamente.
Clicar no X no canto inferior direito da janela principal abrirá  o botão flutuante da área de trabalho. Clicar 2x neste botão abrirá a janela principal. Você pode reposicioná-lo arrastando o botão para qualquer posição na área de trabalho.

O botão flutuante serve para você ativar rapidamente o programa quando trabalha com outras janelas. Para abrir a janela principal, clique duas vezes sobre ele. Para fechar definitivamente o programa, clique com o botão direito. Para mover o botão, clique e arraste para a posição desejada.
